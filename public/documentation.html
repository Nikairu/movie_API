Documentation

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.


REST API documentation
REQUEST - Business Logic URL HTTP Method Request Data Format Response Data Body Format
Return a list of All movies to the user /movies GET None A JSON object holding data about all movies
Return data (description, genre, directors, Producers, etc) about a single movie by title to the user /movies/:[Title] GET None 
				
Example:
{
	"title": "Interstellar",
	"Producers": [
		"Emma Thomas",
		"Christopher Nolan",
		"Lynda Obst"
	],
	"Directors": [
		"Christopher Nolan"
	],
	"genres": [
		"Science fiction",
		"drama"
	]
}
Return data about a genre (description) by name. movies/Genre/:[Name] GET None A JSON object holding data about a genre (descriptio) by name.
Return data about a director (bio, birth year, death year) by name /movies/directors/:[Name] GET None Example: 
			
{
	"name": "Christopher Nolan",
	"bio": "Christopher Edward Nolan CBE (/ˈnoʊlən/; born 30 July 1970) is a British-American film director, producer and...
	"birth_year": "1970",
	"death_year": ""
}
Allow new users to register /users POST Json containing at least the "name" property Example: 

{
	"name": "lenidiogo",
	"password": "123+Qwe."
}
Example: 

{
	"name": "lenidiogo",
	"password": "123+Qwe.",
	"id": "24b994f6-8ed0-4889-9659-af983953773e"
}
Allow users to update their user info (username, password, email, date of birth) /users/:[Name] PUT Json containing at least an existing username. 

{
	"name": "lenidiogo",
	"password": "123+Qwe.",
	"birth_date": "11/01/1990"
}
A text message indicating whether the user's account has succesfully changed.
Allow users to add a movie to their list of favorites /users/:[Name]/addFavouriteMovie POST A JSON object holding data about the movie to add to the user's list of favorite movies: 

{
	"title":"Evangelion: 1.0"
}
A text message indicating whether the user's favorite movie was succesfully added to their list.
Allow users to remove a movie to their list of favorites /users/:[Name]/removeFavorite/:favoriteName DELETE None A text message indicating whether the movie was succesfully removed from the users' favorite-list.
Allow existing users to deregister (remove a user by Username) /users/[Username]/ DELETE None A text message indicating whether the user's account was succesfully removed.
