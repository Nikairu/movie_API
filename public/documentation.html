Documentation

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.


REST API documentation
REQUEST - Business Logic URL HTTP Method Request Data Format Response Data Body Format
Return a list of All movies to the user /movies GET None A JSON object holding data about all movies
Return data (description, genre, directors, Producers, etc) about a single movie by title to the user /movies/:[Title] GET None 
				
Example:
{
	"title": "Interstellar",
	"Producers": [
		"Emma Thomas",
		"Christopher Nolan",
		"Lynda Obst"
	],
	"Directors": [
		"Christopher Nolan"
	],
	"genres": [
		"Science fiction",
		"drama"
	]
}
Return data about a genre (description) by name. movies/genres/:[Name] GET None Example: 

{
    "Name": "Comedy",
    "Description": "A comedy film is a category of film in which the main emphasis is on humor."
}
Return data about a director (bio, birth year, death year) by name /movies/directors/:[Name] GET None Example: 
			
{
	"name": "Christopher Nolan",
	"bio": "Christopher Edward Nolan CBE (/ˈnoʊlən/; born 30 July 1970) is a British-American film director, producer and...
	"birth_year": "1970",
	"death_year": ""
}
Allow new users to register /users POST Json containing at least the "Username", "Password" and "Email" properties Example: 

{
	"Username": "lenidiogo",
	"Password": "123+Qwe.",
	"Email": "lenidiogo@lenidiogo.com"
}
Example: 

{
	"Username": "lenidiogo",
	"Password": "123+Qwe.",
	"Email": "lenidiogo@lenidiogo.com",
	"_id": "24b994f6-8ed0-4889-9659-af983953773e"
}
Allow users to update their user info (username, password, email, date of birth) /users/:[Name] PUT Json containing at least an existing username, password and email. 

{
    "Username": "lenidiogo",
    "Password": "123+Qwe",
    "Email": "lenidiogo@lenidiogo.com",
    "Birthday": "11/01/1990"
}
Json containing updated user info 

{
	"Username": "lenidiogo",
	"Password": "123+Qwe",
	"Email": "lenidiogo@lenidiogo.com",
	"Birthday": "11/01/1990"
}
Allow users to add a movie to their list of favorites /users/:Username/Movies/:MovieID POST None Json containing updated user info 

{
	"Username": "lenidiogo",
	"Password": "123+Qwe",
	"Email": "lenidiogo@lenidiogo.com",
	"Birthday": "11/01/1990"
	"FavoriteMovies":["7d6f564d6d3d57d9d2eh"]
}
Allow users to remove a movie to their list of favorites /users/:Username/Movies/:MovieID DELETE None Json containing updated user info 

{
	"Username": "lenidiogo",
	"Password": "123+Qwe",
	"Email": "lenidiogo@lenidiogo.com",
	"Birthday": "11/01/1990"
	"FavoriteMovies":["7d6f564d6d3d57d9d2eh"]
}
Allow existing users to deregister (remove a user by Username) /users/[Username]/ DELETE None A text message indicating whether the user's account was succesfully removed.
